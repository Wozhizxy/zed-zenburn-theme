# Zenburn & Pale Fire for Zed

Dark color themes for the [Zed](https://zed.dev) editor.

- **Zenburn** — based on the classic [Zenburn](https://kippura.org/zenburnpage/) Vim theme by Jani Nurminen
- **Pale Fire** — based on the [Pale Fire](https://marketplace.visualstudio.com/items?itemName=matklad.pale-fire) VS Code theme by matklad

## Install

Search "Zenburn" in Zed's extension marketplace, or clone this repo and use "Install Dev Extension".
